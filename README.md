# Серверна частина курсової роботи

## Опис
Даний проєкт - серверна частина курсової.  
Проєкт реалізує REST API для обробки HTTP-запитів, взаємодії з клієнтською частиною та роботи з даними.

Застосунок може використовуватися як основа для системи тестування.

---

## Використані технології
- Node.js  
- Express.js  
- npm  
- Morgan  
- MongoDB / SQLite / JSON-файли  
- Postman  

---

## Встановлення та запуск
1. Встановити залежності:

```npm install```

3. Запустити сервер:

```npm run dev```
